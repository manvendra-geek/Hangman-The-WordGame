# 🎯 Hangman Game (Python)

A simple command-line implementation of the classic **Hangman word guessing game** using Python.

---

## 📌 Overview
This project is a word guessing game where the player tries to guess a hidden word within a limited number of attempts.

---

## 🎮 Game Rules
- The number of guesses allowed = **length of the word**
- Each guess (correct or incorrect) reduces the remaining chances by 1
- After each guess:
  - ✅ Correct guess → Letter is revealed in its correct position
  - ❌ Incorrect guess → Error message is shown
- Game continues until:
  - Player guesses the word OR
  - All chances are exhausted

---

## 🏆 Winning Condition
- All letters are correctly guessed  
- Remaining chances > 0  

---

## ❌ Losing Condition
- All chances are used before guessing the word  

---

## 🚀 Features
- Interactive command-line gameplay  
- Tracks correct and incorrect guesses  
- Displays word progress after each attempt  
- Simple and beginner-friendly logic  

---

## 🛠️ Tech Stack
- Python
